¿Qué es HTML?

Consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

Estructura Básica

  • HTML
    Representa la raíz de un documento HTML.
  • HEAD
    Información técnica para el navegador
  • BODY
    Contenido que aprecerá en la página web

Lenguaje HTML

Principales

< !DOCTYPE HTML> Doctype o “Declaración del tipo de documento” es una instrucción especial que va al inicio de nuestro documento HTML y que permite al navegador entender qué versión de HTML estamos utilizando.

< head< /head> Inicio de la cabecera, provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

< title>< /title> Aparece en la barra de título del navegador cuando las personas ven la página web. El título se almacena en los marcadores de tu navegador y en los motores de búsqueda cuando indexan tus páginas.

< body>< /body> Contienen atributos que controlan la parte visible del documento. Estos atributos o propiedades son popularmente usados para agregar color y diseño a la página.

Encabezados

Los elementos de encabezado permiten especificar que ciertas partes del contenido son encabezados, o subencabezados del contenido.

Imágenes

Se incrusta una imagen en la página, en la posición en que aparece. Lo logra a través del atributo src (source), el cual contiene el path (ruta o ubicación) de tu archivo de imagen.

Listas 2.0

Desordenadas: son aquellas en las que el orden de los items no es relevante, Estas son encerradas en un elemento < ul>(unordered list).

Listas

Ordenadas: son aquellas en las que el orden sí es relevante, como en una receta. Estas son encerradas en un elemento < ol>(ordered list).

Vínculos

Para implementar un vínculo, necesitas usar un vínculo simple — — la a es la abreviatura de la palabra inglesa «anchor» («ancla»).

Anidamiento

Los atributos contienen información adicional acerca del elemento, la cual no quieres que aparezca en el contenido real del elemento.

Elementos Vacíos

Algunos elementos no poseen contenido, y son llamados elementos vacíos como

Vocabulario

< br/> Establece dónde comienza o empieza a tener efecto el elemento en este caso, dónde es el comienzo del párrafo.

/ Establece dónde termina el elemento en este caso dónde termina el párrafo.

< p> Si quieres especificar que se trata de un párrafo, podrías encerrar el texto con la etiqueta de párrafo (apertura del elemento).

< muy> Si, quieres resaltar una palabra del texto.

< strong> Significa que dicha palabra se debe enfatizar

< image> Elemento vacío donde se colocan las imágenes de la página web.
